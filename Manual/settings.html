Game Settings

You can change the game settings by tapping "Settings" on the Main Menu. Some options will only take effect next time a new match is started.

Players

Here you can enter each player's names and select the AI difficulty level. Note that "World Class" and "Grandmaster" are significantly slower than the other levels.

Match Length

Select the default length of new matches. To play a money session, select "$".

Use Doubling Cube

If enabled, allows players to raise the stakes of a game under certain conditions.

Use Jacoby Rule

Gammons and Backgammons count only as a single game if neither player offered a double during the game. Only used in money sessions.

Use Crawford Rule

When either player comes within one point of winning the match, the following game is played without a doubling cube.

Enable Sounds

Use this option if you want to hear sound effects when certain events occur.

Show Target Help

Display small check marks next to valid points when moving checkers.

Clockwise Movement

Select the direction that Player 1 will move checkers.
